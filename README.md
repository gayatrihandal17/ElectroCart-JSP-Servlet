# ElectroCart – Electronics Shopping Web Application

ElectroCart is a Java-based web application developed using JSP and Servlets that simulates an online electronics shopping system.

## Technologies Used
- Java
- JSP & Servlets
- HTML, CSS
- Apache Tomcat

## Features
- User login with session management
- Electronics product listing
- Add to cart functionality
- Cart and payment pages

## Project Structure
src/
├── main/java/xyz
│ ├── LoginServlet.java
│ ├── AddToCartServlet.java
│ └── CheckServlet.java
└── main/webapp
├── index.jsp
├── login.jsp
├── product.jsp
├── cart.jsp
├── payment.jsp
└── style.css

## How to Run
1. Import project into Eclipse
2. Configure Apache Tomcat server
3. Run `index.jsp`
